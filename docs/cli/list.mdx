---
title: "list"
description: "Show project configuration"
---

Displays the current project's configuration from `clawup.yaml`, including stack name, agent count, region, and instance type.

## Usage

```bash
clawup list [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--json` | Output as JSON |

## Examples

```bash
# List configs
clawup list

# JSON output
clawup list --json
```
