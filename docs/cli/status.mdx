---
title: "status"
description: "Show the status of your deployed agents"
---

Displays the current status of all agents in your team from Pulumi stack outputs.

## Usage

```bash
clawup status [options]
```

## Options

| Option | Description |
|--------|-------------|
| `--json` | Output as JSON |
| `--local` | Show status of local Docker containers |

## Examples

```bash
# Show status table
clawup status

# JSON output for scripting
clawup status --json

# Show local Docker container status
clawup status --local
```
